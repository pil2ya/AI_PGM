# Python 연산자(Operator) 분류

| 연산자 종류 | 연산자 | 설명 | 예시 |
|:-----------|:-------|:-----|:-----|
| **단항 연산자** | `+`, `-`, `~` | 부호 표시(+), 부호 반전(-), 비트 반전(~) | `-5`, `+3`, `~5` |
| **산술 연산자** | `**`, `*`, `/`, `//`, `%`, `+`, `-` | 거듭제곱, 곱셈, 나눗셈, 몫, 나머지, 덧셈, 뺄셈 | `2**3`, `10//3`, `10%3` |
| **Shift 연산자** | `<<`, `>>` | 비트를 왼쪽 또는 오른쪽으로 이동 | `4 << 1`, `8 >> 2` |
| **비트 연산자** | `&`, `^`, `\|` | 비트 AND, XOR, OR 연산 | `5 & 3`, `5 ^ 3`, `5 \| 3` |
| **관계 연산자** | `<`, `<=`, `>`, `>=`, `==`, `!=`, `is`, `is not`, `in`, `not in` | 값 비교, 객체 동일성 비교, 포함 여부 확인 | `a == b`, `x in list` |
| **논리 연산자** | `not`, `and`, `or` | 논리 부정, 논리 AND, 논리 OR | `not a`, `a and b` |
| **할당 연산자** | `=` | 변수에 값을 저장 | `a = 10` |
| **복합 할당 연산자 (산술)** | `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=` | 산술 연산 후 결과를 변수에 저장 | `a += 5` → `a = a + 5` |
| **복합 할당 연산자 (Shift)** | `<<=`, `>>=` | Shift 연산 후 결과를 변수에 저장 | `a <<= 2` → `a = a << 2` |
| **복합 할당 연산자 (비트)** | `&=`, `\|=`, `^=` | 비트 연산 후 결과를 변수에 저장 | `a &= 3` → `a = a & 3` |

## 한눈에 보는 연산자 분류

| 분류 | 연산자 |
|:-----|:-------|
| **단항 연산자** | `+`, `-`, `~` |
| **산술 연산자** | `**`, `*`, `/`, `//`, `%`, `+`, `-` |
| **Shift 연산자** | `<<`, `>>` |
| **비트 연산자** | `&`, `^`, `\|` |
| **관계 연산자** | `<`, `<=`, `>`, `>=`, `==`, `!=`, `is`, `is not`, `in`, `not in` |
| **논리 연산자** | `not`, `and`, `or` |
| **할당 연산자** | `=`, `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`, `<<=`, `>>=`, `&=`, `^=`, `\|=` |

---

### 참고
- `+=`, `-=`, `*=`, `/=` 등은 **산술 연산자와 할당 연산자가 결합**된 복합 할당 연산자입니다.
- `<<=`, `>>=`는 **Shift 연산자와 할당 연산자가 결합**된 복합 할당 연산자입니다.
- `&=`, `|=`, `^=`는 **비트 연산자와 할당 연산자가 결합**된 복합 할당 연산자입니다.
- `~=`는 존재하지 않습니다. `~`는 단항 연산자이므로 복합 할당 형태가 없습니다.
